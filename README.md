# ta509
Improved input selection board

## Purpose
Intended for the **TuchAudio 309 _Integrated Amplifier_**, together with:

- `TuchAudio 101b` Phono Pre-Amp.
- `TuchAudio 191` Moving-Coil preamp (optional).
- `TuchAudio 408n` simple Volume & Balance Control.
- `TuchAudio 209` Power Stage.
- `TuchAudio PSUÂ·09` Power Supply.

## Options
Besides the use of the Moving-Coil pre-amp `191`, which provides a **_switched_ MM/MC Phono input**,
the `408n` board might be replaced by the `TuchAudio 409n`, which adds **Tone controls** (Baxandall-type).

## Inputs
- **Phono with RIAA equalisation:** both DIN and RCA connectors, 3.6 mV/110 kOhm (optional: _Moving Coil_ input via the `191` board)
- **Line input:** both DIN and RCA connectors, 180 mV/900 kOhm (suitable for _ceramic_ cartridges)
- **Tape Playback:** thru RCA (180 mV/47 k) and DIN (0.48 V/16.5 k) connectors.

## Outputs
- Line out: 180 mV (intended for a suitable poweramp, like **TuchAudio 209** thru a `408/409` _volume_ board).
- Tape Rec (RCA): 180 mV
- Tape Rec (DIN): 5.3 mV over 10 kohm
